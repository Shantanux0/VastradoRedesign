# VastradoRedesign

Modern redesign of Vastrado e-commerce platform celebrating Indian heritage.

## Setup

```bash
npm install
npm run dev
```

## Deploy

### Vercel (Recommended)
1. Push to GitHub
2. Import repository at [vercel.com](https://vercel.com)
3. Deploy automatically

### Netlify
1. Push to GitHub
2. Import repository at [netlify.com](https://netlify.com)
3. Build command: `npm run build`
4. Publish directory: `dist`

### GitHub Pages
Automatic deployment via GitHub Actions on push to main branch.

## Tech Stack

- React + TypeScript
- Vite
- Framer Motion
- React Router
